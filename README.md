# Tchat-At
Tchat server with MongoDB and NodeJS

# Client
```
npm install
npm start
```

Individual install
```
npm install --save socket.io-client
npm install vue
```

# Server
```
npm install
nodemon index.js
```

Individual install
```
npm install --save socket.io
npm install --save-dev nodemon
npm install --save express
npm install --save cors
```
